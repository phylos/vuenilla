# Tabs

## Usage

```html
<v-tabs>
  <v-tab title="One" :selected="true">
    <p>Tab content for first tab.</p>
  </v-tab>
  <v-tab title="Two">
    <p>Tab content for second tab.</p>
  </v-tab>
  <v-tab title="Three">
    <p>Tab content for third tab.</p>
  </v-tab>
</v-tabs>
```
